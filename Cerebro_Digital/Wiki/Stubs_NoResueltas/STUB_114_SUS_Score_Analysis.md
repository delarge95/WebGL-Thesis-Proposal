# SUS Score Analysis

## Proposito

Interpreta el puntaje SUS para convertir una cifra agregada en una lectura util sobre usabilidad percibida y comparacion entre versiones.

## Lectura minima

- Ubicar el valor frente a una escala de referencia.
- Diferenciar percepcion global de problemas concretos.
- Marcar tendencias por version o iteracion.
- Apoyar decisiones de remediacion UX.

## Criterios

1. El analisis debe explicar como se calculo el valor.
2. La comparacion debe usar el mismo contexto de sesion.
3. No sobrerreaccionar a cambios pequeños sin soporte adicional.
4. Relacionar el valor con hallazgos cualitativos.
