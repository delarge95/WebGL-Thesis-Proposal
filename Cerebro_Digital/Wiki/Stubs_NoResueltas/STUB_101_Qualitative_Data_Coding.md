# Qualitative Data Coding

## Proposito

Estructura el material cualitativo para poder extraer temas, patrones y recomendaciones sin perder trazabilidad a la fuente.

## Pautas

- Asignar codigos consistentes.
- Mantener relacion con citas originales.
- Agrupar sin perder matices.
- Distinguir dato y interpretacion.

## Criterios

1. El sistema de codificacion debe ser repetible.
2. Los codigos deben poder revisarse por otra persona.
3. La nota debe explicar la unidad de analisis.
4. Cualquier cambio metodologico debe quedar reflejado.
