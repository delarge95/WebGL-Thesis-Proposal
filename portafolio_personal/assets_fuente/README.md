# Assets Fuente para Portafolio

Carpeta de staging para producir y clasificar el material visual derivado del proyecto.

## Regla operativa

- Esta carpeta no reemplaza assets originales del proyecto.
- Aquí sólo deben entrar capturas, clips, diagramas o exportaciones ya pensadas para portafolio.
- Conviene guardar nombres claros y consistentes para no perder relación con cada breakdown.

## Subcarpetas

### `heroes/`

- Portadas principales.
- Banners para perfil.
- Thumbnails de proyecto o reel.

### `screenshots/`

- Grids de view modes.
- Comparativas antes/después.
- Capturas del visor con anotaciones.

### `clips/`

- Recorridos del visor.
- Exploded view.
- Corte transversal.
- Clips cortos para LinkedIn o reel.

### `diagramas/`

- Diagrama de pipeline CAD→WebGL.
- Diagrama simple de arquitectura o flujo de interacción.
- Láminas de proceso resumidas.
