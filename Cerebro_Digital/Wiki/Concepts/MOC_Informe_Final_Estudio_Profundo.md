# MOC Informe Final Estudio Profundo

Este mapa organiza una variante pedagogica del informe final. No reemplaza el documento formal; funciona como capa de estudio, defensa y explicacion profunda para entender cada modulo del trabajo de grado.

## Como usarlo

1. Lee primero [[INF_EST_00_Guia_Defensa_y_Lectura]] para entender la logica del paquete.
2. Usa el indice puente [[VARIANTE_ESTUDIO_OBSIDIAN]] para saltar desde la estructura del informe hacia cada modulo de explicacion.
3. Revisa [[INF_EST_91_Preguntas_Dificiles_Defensa]] antes de preparar la sustentacion oral.
4. Ensaya la presentacion con [[INF_EST_51_Guion_Completo_Sustentacion_30min]].
5. Usa [[INF_EST_90_Glosario_Global]] cuando un termino del informe bloquee la comprension.

## Criterio pedagogico

Cada modulo intenta responder cuatro preguntas:

- Que dice esta seccion del informe.
- Por que importa para la tesis.
- Que conceptos debe entender alguien no especialista.
- Que preguntas podria hacer un jurado y como responderlas con rigor.
