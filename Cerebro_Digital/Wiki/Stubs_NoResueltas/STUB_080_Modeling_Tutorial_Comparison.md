# Modeling Tutorial Comparison

## Proposito

Compara tutoriales de modelado para decidir que tecnicas, atajos o criterios conviene adoptar en la produccion propia.

## Variables de comparacion

- Claridad pedagogica.
- Fidelidad al objetivo.
- Aplicabilidad al pipeline.
- Riesgo de reproducir tecnicas no compatibles.

## Criterios

1. La comparacion debe ser concreta, no una lista de preferencias.
2. Debe servir para elegir un metodo, no solo para recopilar recursos.
3. Cada tutorial debe evaluarse por lo que aporta al flujo.
4. La nota debe indicar si una tecnica se adopta o se descarta.
