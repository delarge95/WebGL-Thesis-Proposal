# Unity Observer

This skill reads the telemetry generated by `AntigravityBridge.cs` inside Unity. It allows the agent to "see" if the project is compiling, playing, or throwing errors.

## Usage

Use this skill to check the health of the Unity Editor session.

## Capabilities

1.  **Check Editor State**: Is Unity playing? Compiling? specific scene?
2.  **Read Recent Errors**: Fetches the last 5 errors/exceptions from the session.
3.  **Clear Logs**: Resets the error log.

## How to Run

Execute `check_status.ps1` to get a snapshot of the editor.
