# Task Timing Analysis

## Proposito

Estudia cuanto tarda una persona en completar una tarea para localizar pasos lentos, confusos o redundantes dentro del flujo.

## Variables utiles

- Tiempo total por tarea.
- Tiempo por subtarea.
- Numero de errores o retrocesos.
- Diferencias entre usuarios expertos y novatos.

## Criterios

1. El analisis debe indicar la tarea y el escenario medido.
2. Los tiempos deben compararse con el mismo criterio de inicio y fin.
3. Las conclusiones deben distinguir lentitud estructural de aprendizaje inicial.
4. Cualquier outlier relevante debe quedar explicado.
