# Combined Validation Report

## Proposito

Integra los hallazgos dispersos de validacion para obtener una lectura unica de estado, riesgos y remediaciones prioritarias.

## Contenido minimo

- Resumen ejecutivo.
- Hallazgos por categoria.
- Riesgos prioritarios.
- Acciones recomendadas y responsables.

## Criterios

1. El informe debe evitar duplicar detalle ya capturado en notas hijas.
2. Las conclusiones deben apuntar a evidencia enlazable.
3. Debe dejar claro que piezas ya fueron cerradas y cuales siguen abiertas.
4. La version combinada debe servir como entrada de decision para el siguiente bloque.
