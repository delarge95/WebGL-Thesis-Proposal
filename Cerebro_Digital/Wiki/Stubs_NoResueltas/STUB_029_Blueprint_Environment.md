# Blueprint Environment

## Proposito

Define el entorno necesario para trabajar en modo blueprint con una lectura tecnica clara y consistente.

## Condiciones de uso

- Alto contraste y contornos legibles.
- Contexto suficiente del ensamblaje.
- Compatibilidad con selection y inspect.
- Coste controlado al activar el modo.

## Criterios

1. El entorno debe facilitar inspeccion, no decoracion.
2. Debe ser estable entre escenas o versiones.
3. Cualquier efecto adicional debe ser justificado.
4. Debe quedar claro como se diferencia del wireframe.
