```latex
\subsection{Procedimiento de Análisis de Datos}

El análisis de datos sigue un enfoque cuantitativo con componentes descriptivos e inferenciales.

\subsubsubsection{Análisis Descriptivo}

\textit{System Usability Scale (SUS)}\\
El puntaje SUS se calcula mediante la fórmula:
$$SUS = 2.5 \times \sum_{i=1}^{10} s_i$$

Donde $s_i$ representa el puntaje ajustado de cada ítem:
\begin{itemize}
    \item Para ítems impares (1, 3, 5, 7, 9): $s_i = Valor_{respondido} - 1$
    \item Para ítems pares (2, 4, 6, 8, 10): $s_i = 5 - Valor_{respondido}$
\end{itemize}

Interpretación de puntajes (Bangor et al., 2009):
\begin{itemize}
    \item 0-50: Inaceptable (peor que el 15\% de sistemas)
    \item 51-67: Aceptable (promedio de la industria)
    \item 68-80: Bueno (por encima del promedio)
    \item 81-100: Excelente (entre el mejor 3\%)
\end{itemize}

\textit{NASA-TLX (Raw)}\\
El puntaje global se calcula como:
$$NASA-TLX_{score} = \frac{1}{6} \sum_{d=1}^{6} D_d$$

Donde $D_d$ es el puntaje de cada una de las seis dimensiones (0-100): Demanda Mental, Demanda Física, Demanda Temporal, Rendimiento, Esfuerzo, Frustración.

Interpretación:
\begin{itemize}
    \item 0-20: Muy baja carga
    \item 21-40: Baja carga (meta del proyecto)
    \item 41-60: Carga moderada
    \item 61-80: Alta carga
    \item 81-100: Muy alta carga
\end{itemize}

\subsubsubsection{Análisis Inferencial}

Se aplicarán las siguientes pruebas estadísticas:
\begin{enumerate}
    \item \textbf{Estadísticos descriptivos}: Media, desviación estándar, mediana, percentiles 25/50/75 para cada variable.
    \item \textbf{Prueba de normalidad}: Shapiro-Wilk o Kolmogorov-Smirnov para determinar la distribución de los datos.
    \item \textbf{Comparación de medias}: 
    \begin{itemize}
        \item Si distribución normal: t-test pareado para muestras relacionadas (mismos usuarios comparan 3D vs 2D).
        \item Si distribución no normal: Prueba de Wilcoxon para datos pareados.
    \end{itemize}
    \item \textbf{Tamaño del efecto}: Coeficiente d de Cohen para determinar la magnitud práctica de las diferencias.
    \item \textbf{Nivel de significancia}: $\alpha = 0.05$ (95\% de confianza).
\end{enumerate}

\subsubsubsection{Tablas de Resultados Esperadas}

Se construirán las siguientes tablas para la presentación de resultados:
\begin{enumerate}
    \item \textbf{Tabla 1}: Estadísticos descriptivos (N, Media, DE, Mín, Máx) por variable.
    \item \textbf{Tabla 2}: Matriz de correlaciones entre las 6 dimensiones del NASA-TLX.
    \item \textbf{Tabla 3}: Comparación de medias (3D vs 2D) con valores p y tamaño del efecto.
    \item \textbf{Tabla 4}: Intervalos de confianza del 95\% para las diferencias de medias.
\end{enumerate}
```
