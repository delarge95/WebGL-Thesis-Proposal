## 🎯 Quick Start

### Step 1: Review Design System
Read [DESIGN_SYSTEM.md](./DESIGN_SYSTEM.md) to understand:
- Color palette (#0A0E17, #3B82F6, #10B981)
- Typography (Space Grotesk, Inter, JetBrains Mono)
- Layout grid (12-column, 80px margins)
- Component library

### Step 2: Generate with Kimi K2
1. Open Kimi K2 Nano Banana Pro
2. Copy the **Master Prompt** from [KIMI_PROMPT.md](./KIMI_PROMPT.md)
3. Generate the 20-slide presentation
4. Export as PPTX

### Step 3: Add Custom Assets
Use [ASSETS_REQUIREMENTS.md](./ASSETS_REQUIREMENTS.md) to:
- Download fonts (Space Grotesk, Inter, JetBrains Mono)
- Get icons from Phosphor/Lucide (32 icons needed)
- Create diagrams from templates
- Replace placeholder images

### Step 4: Final Review
- Check contrast accessibility
- Verify text limits (max 50 words/slide)
- Export PDF backup
