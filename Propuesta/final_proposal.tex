% --- RESUMEN ---
\section*{Resumen}
\phantomsection
\addcontentsline{toc}{section}{Resumen}
\setlength{\parindent}{0pt} % No indent for Resumen
La documentaci\'on de hardware t\'ecnico complejo, como drones y otros sistemas electromec\'anicos multicomponente, enfrenta un desaf\'io persistente de comunicaci\'on espacial cuando depende de im\'agenes 2D y documentos PDF est\'aticos. Este proyecto aborda esa brecha mediante el dise\~no, desarrollo y evaluaci\'on de un prototipo web 3D interactivo basado en Unity Web para la visualizaci\'on t\'ecnica, inspecci\'on y an\'alisis visual del ensamblaje del dron Holybro X500~V2. El marco te\'orico integra teor\'ia de la carga cognitiva como lente interpretativa, interacci\'on humano-computador en entornos 3D y principios de renderizado f\'isicamente basado. Metodol\'ogicamente, la propuesta se estructura como una investigaci\'on aplicada con enfoque mixto y predominio cualitativo-formativo, enmarcada en Design Science Research y prototipado iterativo. La evaluaci\'on contempla pruebas de rendimiento bajo condiciones controladas, registro de desempe\~no en tareas, estimaciones descriptivas de usabilidad mediante SUS, medici\'on de carga de trabajo percibida con NASA-TLX por condici\'on y observaci\'on \textit{Think-Aloud} sobre tareas representativas contrastadas con un soporte 2D de referencia. El prototipo se concibe para ejecutarse en navegadores web compatibles de escritorio y m\'oviles, reconociendo que la compatibilidad efectiva depende del navegador, su versi\'on, la memoria disponible y el nivel de optimizaci\'on del build. Como resultado esperado, el proyecto busca documentar las condiciones t\'ecnicas y metodol\'ogicas bajo las cuales un procedimiento de optimizaci\'on y visualizaci\'on 3D web resulta operativamente viable en contexto acad\'emico y de ingenier\'ia.

\hspace{1.27cm}\textit{\textbf{Palabras clave:}} WebGL, Unity, WebAssembly, Renderizado F\'isicamente Basado (PBR), Visualizaci\'on 3D Interactiva, Optimizaci\'on de Assets 3D, Carga de Trabajo Percibida, Interacci\'on Humano-Computador, Technical Art, Hardware Complejo.

% --- ABSTRACT ---
\section*{Abstract}
\phantomsection
\addcontentsline{toc}{section}{Abstract}
\setlength{\parindent}{0pt} % No indent for Abstract
Technical communication around complex hardware, such as multicomponent drones and related electromechanical systems, still faces substantial spatial interpretation challenges when it relies on static 2D images and PDF documents. This project addresses that gap through the design, development, and evaluation of an interactive 3D web prototype based on Unity Web for technical visualization, inspection, and visual analysis of the Holybro X500~V2 assembly. The theoretical framework combines cognitive load theory as an interpretive lens, human-computer interaction in 3D environments, and physically based rendering principles. Methodologically, the proposal is framed as applied research with a mixed-method approach and qualitative-formative predominance, supported by Design Science Research and iterative prototyping. Validation includes performance testing under controlled conditions, task-performance recording, descriptive usability estimation through SUS, condition-specific perceived workload measurement with NASA-TLX, and concurrent Think-Aloud observation using representative tasks contrasted against a 2D reference medium. The prototype is intended for compatible desktop and mobile web browsers, while recognizing that effective compatibility depends on browser version, available memory, and build optimization. The expected outcome is to document the technical and methodological conditions under which a web-based 3D optimization and visualization workflow remains operationally viable in academic and engineering contexts.

\hspace{1.27cm}\textit{\textbf{Keywords:}} WebGL, Unity, WebAssembly, Physically-Based Rendering (PBR), Interactive 3D Visualization, 3D Asset Optimization, Perceived Workload, Human-Computer Interaction, Technical Art, Complex Hardware.
