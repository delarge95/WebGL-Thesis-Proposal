# ProceduralPinsIcon

## Proposito

Define pines visuales procedurales para resaltar puntos de interes sin depender de assets manuales para cada caso.

## Reglas de diseno

1. Mantener legibilidad en distintas escalas.
2. Asociar cada pin a un estado o categoria.
3. Evitar saturacion visual cuando hay muchos pines.
4. Mantener coherencia con iconografia general.
