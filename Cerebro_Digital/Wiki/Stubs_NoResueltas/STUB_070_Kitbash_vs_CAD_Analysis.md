# Kitbash vs CAD Analysis

## Proposito

Compara el enfoque kitbash con el modelado CAD para decidir cuando conviene velocidad de ensamblaje y cuando conviene precision estructural.

## Variables de comparacion

- Control de geometria.
- Reutilizacion de piezas.
- Tiempo de produccion.
- Coherencia para validacion tecnica.

## Criterios

1. La comparacion debe explicitar el objetivo del modelo.
2. No tratar kitbash y CAD como equivalentes sin contexto.
3. Resumir ventajas y limites de cada enfoque.
4. La conclusion debe ser util para decisiones de produccion o documentacion.
