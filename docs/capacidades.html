Visualización de Ingeniería // 01

Visualización Avanzada

La aplicación ofrece hasta **7 modos de renderizado** diferentes para estudiar la estructura geométrica e interna del Holybro X500 V2.

Alterna entre la visualización de la malla poligonal (Wireframe) para verificar las uniones y aristas, el modo Rayos X para inspeccionar componentes internos y un renderizado térmico heurístico que estima puntos calientes del motor.

Visualización de Ingeniería // 02

Interacción Total

Explora a fondo la estructura con el sistema de **despiece explosivo**. Separa las capas de ensamble arrastrando el control en tiempo real.

Puedes hacer clic sobre cualquiera de las capas (Hélices, Electrónica, Chasis, Motores) para aislar la pieza, revelando especificaciones de materiales e instrucciones del manual técnico.

Visualización de Ingeniería // 03

Análisis de Ingeniería

Inicializa simulaciones operativas. Enciende el drone para simular revoluciones (RPM) de hélices y observar la telemetría.

El registrador de datos de ingeniería genera gráficos de carga de batería, temperaturas de bobinados y vibración estructural. Puedes introducir **turbulencia del aire** sobrevolando la gráfica con el cursor para alterar los registros.
