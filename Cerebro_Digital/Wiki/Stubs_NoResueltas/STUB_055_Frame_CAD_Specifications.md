# Frame CAD Specifications

## Proposito

Define las especificaciones CAD del frame para que el modelo sea util como base de ensamblaje, fabricacion y comparacion con variantes.

## Requisitos

- Dimensiones principales y tolerancias.
- Puntos de union y compatibilidad mecanica.
- Espesor, rigidez y zonas criticas.
- Relacion con piezas, brazos y soportes.

## Criterios

1. El CAD debe poder usarse para validacion y no solo para presentacion.
2. Los cambios de version deben quedar trazados.
3. Cualquier simplificacion debe indicar que se perdio.
4. La ficha debe conectar con el material y la estructura del frame.
