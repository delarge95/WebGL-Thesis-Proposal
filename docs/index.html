Capabilities

Model and traceability

Interactive representation of the Holybro X500 V2 with 28 canonical parts, 30 scene anchors and technical traceability by component.

Inspection workflow

Hierarchical selection, hotspots, isolation, bottom sheet and technical sheets with identification, specifications and assembly data.

Analysis and visual modes

Exploded view, cross sections, functional-family filters, reconciled fastener reading and technical visual modes including X-Ray, Solid, Blueprint and heuristic Thermal.

Technology

Unity 6

Project built in Unity 6000.4.3f1 with URP 17.x and WebGL output.

WebGL 2.0

Browser-based 3D rendering without plugins, using a responsive canvas.

WebAssembly

C#/IL2CPP runtime compiled for web execution.

UI Toolkit

Runtime interface with mobile-first layout, bottom sheet and modal controls.

Blender to Unity

CAD-to-runtime pipeline with geometry cleanup, instances and manifests.

Shaders HLSL

X-Ray, Solid, clipping, Blueprint, Thermal and PBR materials adapted to WebGL.

Closing state

Synchronized documentation

The landing links manuals, architecture, modules, shaders, deployment, WebGL configuration, thermal system and instruments.

Traceable results

Performance, SUS, NASA-TLX and Think-Aloud evidence is consolidated in the final report and validation annexes.
