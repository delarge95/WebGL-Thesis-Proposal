# Wolfram Workflow

## Goal

Use WolframAlpha as the external verification layer for thermal equations, engineering math, unit conversions, and sanity checks tied to the drone thermal system.

## Endpoint Decision

### Website

Use the website URL when:

- you want a fast manual check,
- there is no AppID available,
- or the query is exploratory.

Pattern:

```text
https://www.wolframalpha.com/input?i=<url-encoded-query>
```

### Short Answers API

Use the Short Answers API when:

- the expected output is a compact scalar,
- you want a single numeric answer,
- or you are checking a conversion or direct closed-form result.

Base URL:

```text
http://api.wolframalpha.com/v1/result
```

### Full Results API

Use the Full Results API when:

- you need assumptions or pod details,
- you need structured JSON/XML,
- or the query is ambiguous and may need richer output.

Base URL:

```text
http://api.wolframalpha.com/v2/query
```

### LLM API

Use the LLM API when:

- you want a concise computed explanation optimized for an LLM,
- or a compact textual interpretation is more useful than pods.

Base URL:

```text
https://www.wolframalpha.com/api/v1/llm-api
```

## Query Patterns for This Project

### Differential equations

```text
solve dT/dt = (T_eq - T)/tau with T(0)=T_ambient
```

### Conduction

```text
k * A / L
```

Example:

```text
0.2 W/(m K) * 4 cm^2 / 2 mm
```

### Heat capacity

```text
m * cp
```

Example:

```text
0.52 kg * 1000 J/(kg K)
```

### Time-to-temperature sanity checks

```text
time required to heat 0.52 kg with cp 1000 J/(kg K) from 20 C to 45 C at 15 W
```

### Unit conversions

Always verify units explicitly.

Examples:

```text
convert 25 cm^2 to m^2
convert 3 mm to m
convert 0.12 W/(m K) to W/(mm K)
```

## Verification Checklist

Before updating code or docs, confirm:

- the equation matches the intended physical law,
- all units are explicit,
- the result magnitude is plausible,
- and the approximation is labeled as a heuristic if it is not data-backed.

## Current Limitation

If no `WOLFRAM_APP_ID` exists, the workflow still works in manual mode through the WolframAlpha website URL generated by the script.
