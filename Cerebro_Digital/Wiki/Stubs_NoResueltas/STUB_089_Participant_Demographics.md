# Participant Demographics

## Proposito

Describe el perfil de participantes para contextualizar hallazgos, sesgos y alcance de las pruebas.

## Variables utiles

- Rango de experiencia.
- Perfil tecnico o no tecnico.
- Distribucion de edades o roles si aplica.
- Condiciones que afectan la interpretacion.

## Criterios

1. La demografia debe mantenerse anonima cuando corresponda.
2. Debe servir para interpretar resultados.
3. No mezclar datos sensibles innecesarios.
4. La nota debe apoyar la lectura de sesgos en la muestra.
