\subsection*{Apéndice I: Cuestionario SUS}
\addcontentsline{toc}{subsection}{Apéndice I: Cuestionario SUS}

Instrumento de evaluación de usabilidad \textit{System Usability Scale} (Brooke, 1986) adaptado al prototipo. Escala Likert de~5 puntos (1~=~Totalmente en desacuerdo, 5~=~Totalmente de acuerdo).

\begin{enumerate}
    \item Creo que me gustaría utilizar este visor 3D con frecuencia.
    \item Encontré el visor innecesariamente complejo.
    \item Pensé que el visor era fácil de usar.
    \item Creo que necesitaría el apoyo de un experto para poder utilizar este visor.
    \item Encontré que las diversas funciones del visor estaban bien integradas.
    \item Pensé que había demasiada inconsistencia en el visor.
    \item Imagino que la mayoría de las personas aprenderían a usar este visor muy rápidamente.
    \item Encontré el visor muy engorroso de usar.
    \item Me sentí muy confiado usando el visor.
    \item Necesité aprender muchas cosas antes de poder empezar a usar este visor.
\end{enumerate}

\noindent \textbf{Tareas guiadas previas al cuestionario:}
\begin{enumerate}
    \item Navegar libremente por el modelo del dron (órbita, zoom, pan).
    \item Seleccionar la pieza ``Motor Brushless'' y leer su información técnica.
    \item Activar la vista explosionada y explorar la estructura interna.
    \item Cambiar al modo de visualización ``X-Ray'' y observar los componentes internos.
    \item Utilizar el corte transversal en el eje~Y para ver una sección del dron.
    \item Buscar una pieza específica en el catálogo de partes.
\end{enumerate}

\subsection*{Apéndice J: Cuestionario NASA-TLX (Raw)}
\addcontentsline{toc}{subsection}{Apéndice J: Cuestionario NASA-TLX (Raw)}

Instrumento \textit{Raw Task Load Index} (Hart y Staveland, 1988) sin ponderación. Escala de~21 puntos (0--100 en incrementos de~5).

\begin{enumerate}
    \item \textbf{Exigencia Mental.} ¿Qué tan mentalmente exigente fue la tarea? (Muy baja -- Muy alta)
    \item \textbf{Exigencia Física.} ¿Qué tan físicamente exigente fue la tarea? (Muy baja -- Muy alta)
    \item \textbf{Exigencia Temporal.} ¿Qué tanta presión de tiempo sintió? (Muy baja -- Muy alta)
    \item \textbf{Rendimiento.} ¿Qué tan exitoso fue en lograr el objetivo? (Perfecto -- Fallido)
    \item \textbf{Esfuerzo.} ¿Qué tan duro tuvo que trabajar para lograr su nivel de rendimiento? (Muy bajo -- Muy alto)
    \item \textbf{Frustración.} ¿Qué tan inseguro, desalentado, irritado o estresado se sintió? (Muy baja -- Muy alta)
\end{enumerate}
