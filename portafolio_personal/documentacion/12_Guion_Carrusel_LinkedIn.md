# Guion de Carrusel para LinkedIn

## Objetivo

Presentar el proyecto en formato corto, visual y profesional para que un recruiter o hiring manager entienda rápido:

- qué es;
- por qué importa;
- qué tiene de técnico;
- y por qué vale la pena mirar el breakdown largo o el repo.

## Formato recomendado

- 6 slides
- formato cuadrado o vertical corto
- una idea por slide
- texto mínimo, legible y directo

### Título

## Copy sugerido para acompañar el post

Comparto una de las piezas más importantes de mi trabajo reciente: un visor WebGL técnico para el Holybro X500 V2 desarrollado en Unity.

El proyecto combina selección de piezas, modos de visualización analítica, sistema térmico híbrido, saneamiento runtime del modelo importado y tooling de editor para mantener consistencia entre escena, datos y UX.

Lo que más me interesó aquí fue conectar visualización técnica, arquitectura runtime y herramientas reales de soporte, en lugar de tratar el proyecto como una simple demo visual.

Si te interesa el breakdown completo, puedo compartir también la versión larga del caso con pipeline CAD -> Unity -> WebGL, sistema térmico y tooling.
