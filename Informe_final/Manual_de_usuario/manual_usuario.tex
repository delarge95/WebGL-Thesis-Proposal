\section{Introducción}
Bienvenido al prototipo Web 3D interactivo para visualización técnica de drones. Este sistema le permite explorar, analizar y aprender sobre la estructura y componentes de drones de alto rendimiento directamente desde su navegador web, sin necesidad de instalar software adicional.

\subsection{¿Para Quién es Este Manual?}
\begin{itemize}
    \item \textbf{Ingenieros de Ensamblaje:} Exploración del modelo con información técnica y, en futuras versiones, guías paso a paso con herramientas de medición y anotaciones.
    \item \textbf{Estudiantes de Ingeniería:} Exploración interactiva de componentes con información técnica detallada.
    \item \textbf{Técnicos de Mantenimiento:} Vistas de despiece, puntos de conexión y especificaciones de torque.
    \item \textbf{Público General:} Navegación intuitiva para comprender la estructura de drones.
\end{itemize}

\section{Requisitos del Sistema}

\subsection{Hardware Mínimo}
\begin{itemize}
    \item \textbf{Dispositivo:} PC, Laptop, Tablet o Smartphone (gama media-alta)
    \item \textbf{RAM:} 4 GB mínimo, 8 GB recomendado
    \item \textbf{GPU:} Compatible con WebGL 2.0
\end{itemize}

\subsection{Software}
\begin{itemize}
    \item \textbf{Navegador:} Chrome 90+ (recomendado), Firefox 90+, Edge 90+, Safari 15+
    \item \textbf{Conexión:} Banda ancha (5+ Mbps) para carga inicial
\end{itemize}

\section{Modos de Visualización}

El sistema ofrece 7 modos de visualización diferentes, accesibles desde la barra de herramientas inferior:

\begin{enumerate}
    \item \textbf{Realista:} Materiales originales con iluminación PBR
    \item \textbf{Rayos X:} Vista semi-transparente para ver componentes internos
    \item \textbf{Blueprint:} Estilo de plano técnico azul
    \item \textbf{Color Sólido:} Un solo color configurable para todas las piezas
    \item \textbf{Wireframe:} Solo las líneas de la geometría
    \item \textbf{Fantasma:} Versión muy transparente del modelo
    \item \textbf{Térmico:} Simulación de vista térmica
\end{enumerate}

\section{Exploración de Piezas}

\subsection{Seleccionar una Pieza}
\begin{enumerate}
    \item Haga clic sobre cualquier componente del dron
    \item La pieza se resaltará visualmente
    \item Aparecerá el panel de información a la derecha
\end{enumerate}

\subsection{Panel de Información}
Al seleccionar una pieza, verá:
\begin{itemize}
    \item Nombre y tipo de componente
    \item Descripción de su función
    \item Peso y dimensiones
    \item Tipo de material y propiedades
    \item Botones de acción: Ocultar, Aislar, Enfocar
\end{itemize}

\subsection{Catálogo de Piezas}
Acceda al catálogo completo desde el botón "Catálogo" en la barra de herramientas. El visor incluye las 16~piezas del dron Holybro X500~V2:
\begin{itemize}
    \item \textbf{Búsqueda:} Escriba el nombre de cualquier pieza (ej.\ ``Pixhawk'', ``Motor'', ``PDB'')
    \item \textbf{Filtrado por categoría:}
    \begin{itemize}
        \item \textit{Structural} (5 piezas): placas de CF, brazos, tren de aterrizaje, plataforma, rieles
        \item \textit{Electronics} (7 piezas): PDB, módulo de potencia, Pixhawk~6C, GPS, radio, batería, receptor RC
        \item \textit{Propulsion} (3 piezas): motores~×4, ESCs~×4, hélices~×4
        \item \textit{Other} (1 pieza): kit de herrajes y tornillería
    \end{itemize}
    \item \textbf{Visibilidad:} Toggle de ojo para ocultar/mostrar cada pieza
    \item \textbf{Info detallada:} Al seleccionar una pieza del catálogo se muestran peso, dimensiones, material, fabricante, número de parte, función y advertencias de seguridad
\end{itemize}

\section{Vista Explosionada (Despiece)}

\subsection{Activar Vista Explosionada}
\begin{enumerate}
    \item Presione el botón "Explode" en la barra de herramientas
    \item Use el deslizador para controlar el nivel de separación
    \item Las piezas se separan animadamente según su orden de ensamblaje
\end{enumerate}

\subsection{Interactuar en Vista Explosionada}
\begin{itemize}
    \item Las piezas siguen siendo seleccionables
    \item La información técnica permanece accesible
    \item Puede combinar con otros modos de visualización
\end{itemize}

\section{Cortes Transversales}

La herramienta de corte transversal permite ver el interior del dron:
\begin{enumerate}
    \item Presione el botón de tijera en la barra de herramientas
    \item Seleccione el eje de corte (X, Y o Z)
    \item Ajuste la posición del plano de corte con el deslizador
    \item Active "Invertir" para cambiar la dirección del corte
\end{enumerate}

\section{Encendido del Dron (Simulación)}

\textit{Esta funcionalidad se encuentra en desarrollo para la versión actual del prototipo.}

Simule el funcionamiento del dron:
\begin{enumerate}
    \item Presione el botón de encendido (Power)
    \item Observe la secuencia de arranque:
    \begin{itemize}
        \item Las luces cambian de rojo a verde
        \item Las hélices comienzan a girar
        \item Se activan los efectos de partículas
    \end{itemize}
    \item En modo "Volando", el dron simula movimiento de hover
    \item Presione nuevamente para apagar
\end{enumerate}

\section{Configuración y Accesibilidad}

Acceda a Configuración desde el menú:
\begin{itemize}
    \item \textbf{Volumen:} Ajuste audio maestro, efectos y música
    \item \textbf{Escala UI:} Aumente o reduzca el tamaño de la interfaz
    \item \textbf{Alto Contraste:} Active para mejor visibilidad
    \item \textbf{Reducir Movimiento:} Desactive animaciones para sensibilidades
\end{itemize}

\section{Solución de Problemas}

\subsection{El modelo no carga}
\begin{itemize}
    \item Verifique su conexión a internet
    \item Actualice su navegador a la última versión
    \item Visite \url{https://get.webgl.org/} para verificar compatibilidad WebGL
    \item Recargue la página (F5 o Ctrl+R)
\end{itemize}

\subsection{Rendimiento lento}
\begin{itemize}
    \item Cierre otras pestañas y aplicaciones pesadas
    \item En portátiles, conecte a la corriente eléctrica
    \item El sistema ajusta automáticamente la calidad; espere unos segundos
    \item Presione F3 para ver las métricas de rendimiento
\end{itemize}

\subsection{Audio no funciona}
\begin{itemize}
    \item Verifique que el navegador permita audio automático
    \item Haga clic en cualquier parte de la aplicación para activar audio
    \item Revise el volumen en Configuración
\end{itemize}
