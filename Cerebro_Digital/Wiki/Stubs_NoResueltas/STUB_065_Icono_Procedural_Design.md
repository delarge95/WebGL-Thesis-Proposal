# Icono Procedural Design

## Proposito

Diseña iconos generados de forma procedural para estados o categorias, manteniendo consistencia visual y escalabilidad.

## Reglas de diseño

- El icono debe ser legible en pequeño.
- Debe representar una categoria o estado de forma clara.
- El diseño debe ser reproducible y versionable.
- Conviene evitar complejidad innecesaria.

## Criterios

1. La nota debe explicar el metodo procedural usado.
2. Debe conectar con el sistema de categorias.
3. No sacrificar claridad por originalidad.
4. El icono debe mantenerse util en UI y reportes.
