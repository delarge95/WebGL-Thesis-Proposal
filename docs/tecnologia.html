Volver a la Landing
Tecnología de Desarrollo // 01

Unity 6 LTS

El motor gráfico de la aplicación. Configurado bajo el pipeline de renderizado universal (URP) de Unity.

La integración con herramientas de edición en runtime permite controlar transformaciones espaciales en 3D (Traslación, Rotación y Escala) mediante gizmos interactivos configurados en base a vectores cartesianos.

Universal Render Pipeline Gizmos Vectoriales
Tecnología de Desarrollo // 02

WebGL 2.0

Proporciona aceleración por hardware directa en el navegador. La GPU procesa mallas y cálculos matriciales de cámara sin plugins.

La simulación muestra un **Point Cloud (nube de puntos) interactivo** en 3D. Las partículas son rechazadas magnéticamente por el cursor, y puedes ordenarles cambiar de forma en tiempo real.

HTML5 Canvas GPU Point Cloud 3D
Tecnología de Desarrollo // 03

WebAssembly

Permite ejecutar código compilado nativo (C# traducido a C++ e IL2CPP) en el navegador a máxima velocidad.

Al pulsar "COMPILAR", la simulación corre el árbol de compilación del compilador IL2CPP de Unity y grafica el rendimiento final frente a un intérprete Javascript convencional.

IL2CPP Compilation WASM Virtual Machine
Tecnología de Desarrollo // 04

HLSL Shaders

Cálculo de píxeles a nivel GPU. Usados en la aplicación para colorear la malla en modo Rayos X o simular el espectro térmico.

Manipula las variables uniformes del shader de fragmentos (velocidad, frecuencia de onda y distorsión de metaballs) con los controles deslizantes para ver los resultados matemáticos aplicados al instante.

Pixel Shaders Uniform Variables
