# enlaces

## Proposito

Esta nota consolida el uso de `enlaces` como marcador para referencias internas en revisión, útil mientras se decide el nombre final del concepto.

## Criterio

1. Si se usa para arquitectura documental, renombrar a un hub concreto.
2. Si se usa para enlace temporal, mantenerlo solo con trazabilidad.
3. Si no se necesita, eliminarlo al cerrar el bloque correspondiente.
