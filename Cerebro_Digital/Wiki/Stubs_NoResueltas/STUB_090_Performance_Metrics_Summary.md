# Performance Metrics Summary

## Proposito

Agrupa indicadores de rendimiento para detectar cambios relevantes entre iteraciones, configuraciones y escenarios de carga.

## Indicadores utiles

- Tiempo de respuesta.
- Uso de CPU y memoria.
- Tasa de frames o estabilidad visual.
- Costo de operaciones criticas durante pruebas.

## Criterios

1. La sintesis debe preservar la fuente de cada indicador.
2. No mezclar metricas de distinta naturaleza sin etiquetarlas.
3. Resaltar cambios significativos y no solo valores aislados.
4. Vincular la lectura con el contexto de prueba correspondiente.
